    # tools

    # 翻墙

    主 google cloud (opens new window)，备 rixcloud (opens new window)、阿里云香港和公司翻墙。

    • google cloud 免费，一年到期后可无限续，支持 netflix
    • rixcloud 到期后可能不续了，有点贵，不过老牌服务商，比较稳定，支持 surge 客户端，当时买这个主要是为了看 netflix，后来 google cloud 支持 netflix 后就没必要再买他了
    • 软件方面，Mac 下用 ShadowsocksX-NG-R (opens new window)，iPhone 下用 ShadowRocket (opens new window)（需切美区下载）
    • 通过 Proxifier (opens new window) 实现命令行应用或其他应用翻墙，比如 iTerm2 下执行 npm publish 偶尔会被墙，并且实测下来比 export http_proxy 快
    • 家里的路由器翻墙是用 RT-AC88U (opens new window) + 梅林小宝版固件
    • 电视上看 youtube 和 netflix 可以用 Nvidia Shield TV (opens new window)，我买的美版，据说国版也可刷原生系统

    # Mac 软件

    # 编辑器和 Terminal

    主 Intellij Idea (opens new window)，辅 VSCode (opens new window) 和 Vim。选 Intellij Idea 的原因是无需安装插件就很好用，另外也是没有时间去折腾插件。

    • 字体主 Dank Mono (opens new window)，辅 Operator Mono (opens new window)，看厌了一个换另一个
    • Intellij Idea 使用 material-theme-jetbrains (opens new window)，Theme 选 Material One Dark，字号 16 号，行距 1.2，效果图 (opens new window)
    • Intellij Idea 插件额外装了 File Watchers、GitLink、Import Cost、Prettier 和 Rainbow Brackets
    • Terminal 用 iTerm2 (opens new window) + zsh (opens new window) + oh-my-zsh (opens new window) 的组合，主题是 robbyrussell (opens new window)
    • zsh 的插件开了 git、autojump、brew、git、git-extra、git-flow、git-prompt、git-remote-branch、github、gitignore、history、history-substring-search、iterm2、node、npm、npx、nvm、tig、vscode、yarn
    • iTerm2 里配 Run command... 为 /usr/local/bin/idea \1 (图 (opens new window))，这样 Command + 点击文件路径，就会在 Intellij Idea 里打开

    # 开发辅助

    • SourceTree (opens new window)，git 辅助，由于 git 高级操作命令记不住，就只用借助 UI 了
    • Paw (opens new window)，请求模拟，前后端联调时我会用这个先走一遍
    • Github Desktop (opens new window)，管理 github 仓库的变更和 PR，代替了 SourceTree 的部分工作，可以方便地把别人的 PR checkout 到本地验证
    • Gas Mask (opens new window) ，Hosts 管理
    • ColorSnapper2 (opens new window)，取色工具
    • Charles (opens new window)，抓包用，支持 https
    • Google Chrome (opens new window) + Google Chrome Canary (opens new window) + Firefox (opens new window) + Safari，浏览器，调试用，IE 的测试会借助内网的云测平台

    # 输出

    • Notion (opens new window)，笔记工具，准备逐渐从 Bear 迁到 Notion
    • Bear (opens new window)，笔记工具，颜值高，订阅了 Pro，和手机同步
    • Typora (opens new window)，可以基于目录编辑 Markdown，所以 github 仓库的文档都会用他编辑
    • OmniGraffle (opens new window) + iThoughtsX (opens new window)，画流程图
    • LICEcap (opens new window) ，GIF 录屏工具

    # 输入

    • Reeder (opens new window)，RSS 阅读软件，我的主要信息来源，没有提供 rss 源的我会先在 rsshub.app (opens new window) 上找，再没有就自己写一个 serverless 服务部署在 now@2 上
    • Kiwi for Gmail (opens new window)，Gmail 客户端

    # 效率

    • Alfred (opens new window) + Powerpack (opens new window)，应用启动、粘贴板管理、Workflow、Snippets 管理等
    • Thor (opens new window)，一键直达应用
    • OmniFocus (opens new window) ，任务管理，通过 Omni Sync Server 和 iPhone 同步
    • Karabiner Element (opens new window)，用于把右 Command 和 Capslock 键利用起来 (opens new window)，避免快捷键冲突，简单 note (opens new window)，详见《我的快捷键技巧》 (opens new window)

    # vlog 相关

    • ScreenFlow (opens new window)，视频录制和剪辑
    • Final Cut Pro (opens new window)，视频剪辑
    • RDM (opens new window)，分辨率切换，允许设置未支持的分辨率，比如我会在录屏时设置 720p(hd) 的分辨率
    • KeyCastr (opens new window)，按键显示

    # 其他

    • CleanMyMac 3 (opens new window)，系统清理
    • 1Password (opens new window)，密码管理
    • Bartender 3 (opens new window)，管理系统右上菜单项，隐藏或收起不常用的
    • KeepingYouAwake (opens new window)，可保证系统不自动休眠
    • Softorino YouTube Converter (opens new window)，YouTube 视频下载
    • Downie 3 (opens new window)，通用视频下载
    • Folx 5 (opens new window) + qBittorrent (opens new window) + Motrix (opens new window)，下载工具，Folx 下 http，qBittorrent 下 magnet，Motrix 是 aria2 的封装，可以下百度云盘、115 等 (opens new window)
    • f.lux (opens new window)，调节显示器色温，护眼，尤其是早上起来屏幕实在是刺眼
    • IINA (opens new window)，视频播放
    • snipaste (opens new window)，截图工具，需要标注的时候用
    • Get Plain Text (opens new window)，自动清除粘贴板内容的格式
